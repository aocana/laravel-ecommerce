# Getting Started

### Prerequisites

- [Docker](https://docs.docker.com/get-docker/)
- [Docker Compose](https://docs.docker.com/compose/install/)

### Installation

Clone repository:

```
git clone https://github.com/ocania/laravel-ecommerce.git
cd laravel-ecommerce
```

Create .env file and edit

```
cp .env.example .env
```
